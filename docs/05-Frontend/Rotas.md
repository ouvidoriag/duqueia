# Rotas e Visualização do Frontend — Duque IA

O servidor Node.js expõe os arquivos estáticos de forma direta no navegador:

- `http://localhost:3000/` ou `/chat.html` -> Interface principal do Chatbot.
- `http://localhost:3000/arquitetura.html` -> Exibição técnica do RAG.
- `http://localhost:3000/prompts.html` -> Exposição de prompts da IA.
- `http://localhost:3000/organograma.html` -> Organograma do governo municipal.
